# Random Password Generator - Генератор случайных паролей

**Автор:** Ахметова Дина

## Описание программы

Random Password Generator - это приложение с графическим интерфейсом для создания надежных случайных паролей. Программа позволяет пользователю настраивать параметры генерации, сохранять историю созданных паролей и управлять ею.

### Основные функции:

- **Генерация паролей** заданной длины (от 4 до 32 символов)
- **Выбор типов символов**: цифры, буквы (строчные и прописные), специальные символы
- **Визуальный ползунок** для удобной настройки длины
- **Копирование пароля** в буфер обмена одним кликом
- **История паролей** с отображением даты создания и использованных символов
- **Сохранение истории** в JSON файл (автоматическая загрузка при запуске)
- **Экспорт истории** в отдельный JSON файл
- **Валидация ввода** (проверка длины и выбора символов)
